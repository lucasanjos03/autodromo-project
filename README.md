# 🏎️ Sistema de Gestão de Autódromo

Projeto desenvolvido para a disciplina de Sistemas de Informação.

## 🚀 Tecnologias
- **Java 21** & **Spring Boot 3.5**
- **Docker** & **Docker Compose**
- **PostgreSQL**
- **Lombok** & **Spring Data JPA**

## 🛠️ Como rodar o projeto
1. Entre na pasta `/docker` e rode `docker-compose up -d`.
2. Abra o projeto no IntelliJ e execute a classe `GestaoCorridaApplication`.
